Artwork Embed Demo

Experiment with different ways to embed live artwork on your site.

Option 1: Framed Artwork

Embed the artwork in a contained frame. Great for showcasing pieces in a gallery or section.

"36 Points" by Sage Jenson

<div class="artwork-frame">
  <iframe src="YOUR_ARTWORK_URL" title="Artwork Title"></iframe>
</div>

Option 2: Fullscreen Background

Use the artwork as a living background behind your content. Toggle it on to see the effect.

<!-- Add before page-wrapper -->
<div class="artwork-background">
  <iframe src="YOUR_ARTWORK_URL"></iframe>
</div>

<!-- Add class to body -->
<body class="has-artwork-bg">

Tips:

  • Use content-overlay class on content sections for readability
  • Add dark-overlay to body for dark theme over light artwork
  • Set pointer-events: auto on artwork-background to allow interaction
  • Adjust backdrop-filter: blur() for more/less blur effect

Size Variations

Adjust the frame size using aspect-ratio or fixed dimensions.
