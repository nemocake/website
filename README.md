# con.rad

personal website. vanilla html/css/js.

## setup

1. clone it
2. edit `data/config.json` for content
3. push to github pages

## structure

- `data/config.json` - all site content (collection, writings, etc.)
- `assets/js/config-loader.js` - loads config across pages
- pages are self-contained html files
